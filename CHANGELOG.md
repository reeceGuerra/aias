# Changelog

Rho AIAS uses a lightweight architecture versioning scheme:

- **Major**: fundamental change in structure, philosophy, or artifact relationships
- **Minor**: new mode, command, skill, contract update, or behavioral refinement

## Current Version

The current version of the architecture is **v7.6**.

## Versioning Scheme

| Level | When to bump | Example |
|---|---|---|
| **Major** (v4 -> v5) | Fundamental change in structure, philosophy, or artifact relationships | Redesign two-message pattern, introduce autonomous sub-agents, break backward compatibility |
| **Minor** (v4.6 -> v4.7) | New mode, command, skill, contract update, or behavioral refinement | Add `@devops`, create `/blueprint`, trim `planning.mdc`, add `CONTEXT ENRICHMENT` to modes |
